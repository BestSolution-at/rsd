## Added new built-in types local-time and offset-date-time {{ date: '2026-05-15T10:16Z'}}

We added 2 new built-in types `local-time` and `offset-date-time`.

Beside these we now have full support for:

- generating OpenAPI 3.0 specifications - https://swagger.io/specification/v3/
- full support for message-pack format - https://msgpack.org/

The test suite has been extended to cover almost all possible combinations of these features
